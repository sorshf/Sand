# 🏖️Falling Sand Simulation

Falling sand simulation built using **C++** and **SDL3**.

## Dependencies

- [SDL3](https://www.libsdl.org) – Window rendering
- C++17 or later

## Build

```bash
git clone https://github.com/sorshf/Sand.git
cd Tuner
mkdir build && cd build
cmake ..
make
./bin/Tuner
```
